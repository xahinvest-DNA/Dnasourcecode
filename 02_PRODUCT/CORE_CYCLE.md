# Core Cycle

## Old Cycle
`belief -> attention -> behavior -> result -> reinforcement`

### Example
- Belief: money comes only through strain
- Attention: user notices obstacles, not leverage
- Behavior: user overworks and underprices
- Result: low or unstable income
- Reinforcement: old belief appears confirmed

## Intervention Goal
Replace the self-reinforcing old cycle with a structured new cycle:
`new belief -> new attention target -> new behavior -> new result marker -> new reinforcement`

## Product Responsibility
- Identify the old cycle
- Show the user where the cycle keeps closing on itself
- Offer one admissible new belief
- Convert that belief into one new action
- Record whether a new result marker appeared

## Value Unit
- One completed restructuring cycle
