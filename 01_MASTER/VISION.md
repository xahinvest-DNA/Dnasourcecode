# Vision

## Product Thesis
Build a mini-app trainer that helps a user identify a hidden poverty mechanism, see the self-reinforcing cycle it creates, and complete one restructuring cycle that produces a new belief, a new focus, one bounded action, and a recorded shift.

## Desired User Outcome
- The user can name the hidden mechanism instead of only describing symptoms.
- The user sees how the old cycle keeps repeating.
- The user receives one psychologically admissible reframing rather than an unbelievable opposite statement.
- The user completes one real micro-action.
- The user records evidence of shift in attention, behavior, and result.

## Product Position
- Not therapy
- Not motivation content
- Not an affirmation generator
- Not a generic journaling app
- A structured trainer for internal cycle restructuring

## First Domain
- Money and income
- Focus: hidden poverty mechanism and money-related self-reinforcing cycle
